# HTML and CSS Project

## A fully responsive project built using only HTML and CSS

In this project, I used my newly acquired HTML and CSS skills to transform my Rainy Days design (from the Design Course) into a functional website.

This project demonstrates the following:

- The website closely matches the Rainy Days design created in the Design Course
- Navigation between multiple pages using a navigation menu
- Mimicking certain functionality normally requiring JavaScript (for example, a login page)
- Use of Flexbox and CSS Grid where appropriate
